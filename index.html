部署步骤

1

创建GitHub账户

访问 github.com/signup 注册账户

2

创建新仓库

仓库名格式: yourusername.github.io

3

上传HTML文件

将您的HTML文件上传到仓库（命名为index.html）

4

启用GitHub Pages

仓库设置 → Pages → 选择分支 → 保存

5

访问您的网站

打开浏览器访问: https://yourusername.github.io

首次部署可能需要1-2分钟生效

项目结构示例

最简单的HTML项目只需要一个index.html文件：

yourusername.github.io/
└── index.html

包含CSS/JS资源的项目结构：

yourusername.github.io/
├── index.html
├── styles.css
├── script.js
└── assets/
    ├── image1.jpg
    └── image2.png

使用Git命令行部署

对于开发者，推荐使用Git命令行：

# 创建新仓库目录
$ mkdir yourusername.github.io
$ cd yourusername.github.io

# 初始化Git仓库
$ git init

# 创建HTML文件
$ echo "<!DOCTYPE html>" >> index.html
$ echo "<html><body><h1>Hello World!</h1></body></html>" >> index.html

# 添加文件并提交
$ git add index.html
$ git commit -m "Initial commit"

# 连接到GitHub仓库
$ git remote add origin https://github.com/yourusername/yourusername.github.io.git

# 推送到GitHub
$ git push -u origin main
